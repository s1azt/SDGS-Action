## 📌 提案概要

社内休憩スペース「ネコカフェ」に**エアロバイク型発電機**を設置し、社員が自分の力でスマートフォンを充電できるようにする取り組みです。

運動不足解消と環境貢献を同時に実現し、楽しみながらSDGsに貢献できる仕組みを構築します。

---

## 🎯 解決する3つの課題

### 1. **スマホの充電切れ問題**
- 仕事中にスマホのバッテリーが切れて困る
- 充電したいけどコンセントが見つからない
- 私物充電のコンプライアンスリスク

### 2. **運動不足の問題**
- デスクワーク中心で運動時間が取れない
- 健康診断で運動不足を指摘される
- リフレッシュの方法が限られている

### 3. **ネコカフェの活用不足**
- せっかくの休憩スペースが十分活用されていない
- 景色は良いが、休憩以外の使い方がない
- 社員同士のコミュニケーションの場が少ない

## ⚖️ コンプライアンス対応

### **私物充電の3つのリスク**

#### 1. **法律上のリスク（電気窃盗罪）**
- 刑法245条: 電気は「財物」とみなされる
- 会社の許可なく電気を使うと窃盗に該当する可能性
- **解決:** 自己発電なので法的リスクゼロ

#### 2. **社内規定違反のリスク**
- 私物利用に関する社内規定
- 電力使用のルール
- **解決:** 会社の電力を使わない

#### 3. **マナー・モラルのリスク**
- 「会社のお金で私物充電している」という印象
- 同僚からの視線
- **解決:** 自分で発電した電力を使用

### **追加効果**
- 会社の電力使用量削減（月間約4.7kWh）
- 電気代削減（年間約1,700円）

## 📐 CO2削減量の計算根拠

### **前提条件**
- **1日あたり利用者数:** 5人
- **1人あたり平均利用時間:** 5分
- **対象期間（2月）:** 25日間（2/25時点）
- **1人あたり平均発電量:** 8.3Wh（100W × 5/60h）
- **電力のCO2排出係数:** 0.5 kg-CO2/kWh（日本の電力会社平均）

### **計算プロセス**

#### **ステップ1: 1日あたりの発電量**
```
1日あたりの発電量 = 5人 × 8.3Wh = 41.5Wh
```

#### **ステップ2: 月間総発電量（25日間）**
```
月間総発電量 = 41.5Wh × 25日 = 1,037.5Wh = 1.0375kWh
```

#### **ステップ3: CO2削減量**
```
月間CO2削減量 = 1.0375kWh × 0.5 kg-CO2/kWh = 0.51875 kg-CO2 ≈ 0.5kg
```

#### **年��換算**
```
年間CO2削減量 = 0.5kg × 12ヶ月 = 6kg
```

### **実感イメージ**
- 🍼 ペットボトル約**20本分**のリサイクル効果（月間）
- 🚗 車で約**2.7km**の走行削減に相当（月間）
- 🌳 杉の木**2本分**のCO2吸収量（年間）

## 💬 想定Q&A

### **Q1: CO2削減量2.3kgの計算根拠は？**
**A:** 月間利用者47人×1人あたり100Wh＝4.7kWhの発電量。これに電力のCO2排出係数0.5kg-CO2/kWhをかけると2.35kg、約2.3kgになります。年間では約28kgのCO2削減効果です。

### **Q2: 本当に続けられますか？**
**A:** はい！ランキングやリアルタイム表示で「見える化」することで、ゲーム感覚で楽しめます。義務ではなく「やりたい」仕組みだから続きます。

### **Q3: 費用対効果は？**
**A:** 初期費用5万円で、社員の健康促進と環境貢献を同時実現。数字では測れない「社内コミュニケーション活性化」という副次効果も期待できます。

### **Q4: 設置場所は？**
**A:** ネコカフェの窓際、晴海の景色が見える場所を想定しています。既存のスペースを活用するので、追加スペースは不要です。

### **Q5: 故障やメンテナンスは？**
**A:** 定期的な点検と清��で対応。利用者からのフィードバックを元に改善を続けます。

### **Q6: 私物充電のコンプライアンスについて詳しく教えてください**
**A:** 刑法245条により電気は財物とみなされ、許可なく会社の電源を使うと電気窃盗に該当する可能性があります。このシステムでは自己発電のため、そのリスクが完全に解消されます。さらに会社の電力使用量も削減できます。

### **Q7: 対外的なアピール効果は本当にあるのでしょうか？**
**A:** はい、大きな効果が期待できます。この取り組みは「具体的・ビジュアル映え・ストーリー性」の3拍子が揃っています。採用サイトやSNSでの発信、プレスリリース、CSR報告書への掲載など、多方面で活用できます。特にZ世代の求職者や、ESG投資を重視するステークホルダーへの訴求力が高いと考えています。
