## 📋 配布資料（PRESENTATION_HANDOUT.md）をPDF化

### **手順1: Markdownをブラウザで開く**

1. **Chromeで開く**
   - `/PRESENTATION_HANDOUT.md` ファイルを右クリック
   - 「プログラムから開く」→「Google Chrome」を選択

   **または**
   
   - Chrome拡張機能「Markdown Viewer」をインストール
   - https://chrome.google.com/webstore/detail/markdown-viewer/
   - Markdownファイルをブラウザにドラッグ&ドロップ

### **手順2: 印刷してPDF保存**

1. **印刷メニューを開く**
   - `Ctrl + P`（Windows）
   - `Command + P`（Mac）

2. **設定**
   - **送信先:** 「PDFに保存」
   - **レイアウト:** 縦向き
   - **ページ:** すべて
   - **余白:** 標準
   - **背景のグラフィック:** オン

3. **保存**
   - ファイル名: `SDGs発表資料_配布版.pdf`
   - 「保存」をクリック

**完了！** これで配布資料のPDFができました。

---

## 🖼️ プレゼンスライドをPDF化（スクリーンショット方式）

### **Windows の場合**

#### **手順**

1. **ブラウザでアプリを開く**
   ```
   http://192.168.8.205/wordpress/sdgs-presentation/
   ```

2. **全画面モードにする**
   - `F11` キーを押す（全画面表示）

3. **各スライドをスクリーンショット**
   - スライド1を表示
   - `Windows + Shift + S` を押す
   - 画面全体を選択
   - 自動的にクリップボードにコピーされる

4. **PowerPointに貼り付け**
   - PowerPointを開く
   - 新しいスライドを追加
   - `Ctrl + V` で貼り付け
   - 画像をスライド全体に拡大

5. **繰り返し**
   - スライド2〜13まで同じ作業を繰り返す

6. **PDF保存**
   - 「ファイル」→「エクスポート」→「PDF/XPSの作成」
   - ファイル名: `SDGs発表スライド.pdf`
   - 「発行」をクリック

**完了！** 13ページのPDFスライドができました。

---

### **Mac の場合**

#### **手順**

1. **ブラウザでアプリを開く**
   ```
   http://192.168.8.205/wordpress/sdgs-presentation/
   ```

2. **全画面モードにする**
   - `Control + Command + F`（全画面表示）

3. **各スライドをスクリーンショット**
   - スライド1を表示
   - `Command + Shift + 4` → `Space` を押す
   - カーソルがカメラアイコンに変わる
   - ブラウザウィンドウをクリック
   - デスクトップに画像が保存される

4. **繰り返し**
   - スライド2〜13まで同じ作業を繰り返す

5. **PDFに結合（プレビューアプリ使用）**
   - すべての画像を選択
   - 右クリック→「このアプリケーションで開く」→「プレビュー」
   - サイドバーで全画像を選択（`Command + A`）
   - 「ファイル」→「PDFとして書き出す」
   - ファイル名: `SDGs発表スライド.pdf`
   - 「保存」をクリック

**完了！** 13ページのPDFスライドができました。

## 📱 オンラインツールで簡単PDF化（最も簡単）

### **推奨サービス: Markdown to PDF**

1. **サイトにアクセス**
   - https://www.markdowntopdf.com/

2. **ファイルをアップロード**
   - `/PRESENTATION_HANDOUT.md` をドラッグ&ドロップ

3. **変換**
   - 「Convert to PDF」ボタンをクリック

4. **ダウンロード**
   - PDFファイルが自動ダウンロードされる

**完了！** わずか3ステップでPDF化できました。

## 📊 VS Codeを使う方法（プログラマー向け）

### **手順**

1. **VS Code拡張機能をインストール**
   - 拡張機能: `Markdown PDF` by yzane
   - インストールURL: https://marketplace.visualstudio.com/items?itemName=yzane.markdown-pdf

2. **Markdownファイルを開く**
   - `/PRESENTATION_HANDOUT.md` を開く

3. **PDF出力**
   - 右クリック→「Markdown PDF: Export (pdf)」
   - または `Ctrl + Shift + P` → `Markdown PDF: Export (pdf)`

4. **完了**
   - 同じフォルダに `.pdf` ファイルが自動生成される

**完了！** 1クリックでPDF化できました。

## 💡 プロのTips

### **高品質なPDFを作るコツ**

✅ **スクリーンショットは横1920px以上**  
✅ **印刷設定で「背景のグラフィック」を必ずON**  
✅ **各スライドで2秒待ってからスクリーンショット（アニメーション完了を待つ）**  
✅ **ファイルサイズは3MB以下に抑える（メール添付可能サイズ）**

## 🆘 困ったときは

### **問題: Markdownが文字化けする**
→ ファイルをメモ帳で開いて「UTF-8で保存」し直す

### **問題: PDFのファイルサイズが大きすぎる**
→ https://www.ilovepdf.com/compress_pdf で圧縮

### **問題: 画像がぼやける**
→ スクリーンショットを高解像度（1920x1080以上）で撮る
